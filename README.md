**task 5.1**

Таблица ROSSTAT_SALARY_RU, содержит сведения о средней заработной плате в РФ по регионам на 1 января 2019 года по данным Росстата. Представим ситуацию, что из-за невнимательности операциониста, регионы: Камчатский край, Ивановская область, Хабаровский край оказались не представлены в итоговой сводке. Роль невнимательного операциониста придется исполнить Вам (т.е. нужно удалить соответствующие строки), а далее работать уже с новой выборкой.

Рекомендуем получив выборку воспользоваться RapidMiner или Python для определения статистик и знакомства с инструментами.

Постройте вариационный ряд X(сортировка по неубыванию, нумерация элементов начинается с 1).
- Ввести $Х_4$
- Ввести $Х_{22}$
- Ввести $Х_{45}$
- Найдите выборочное среднее
- Определите выборочную медиану

Во второй таблице представлены данные, по уникальным словам, и числу их вхождений в указанные группыю
Задача - построить модель наивного байесовского классификатора и определить класс,
к которому будет отнесено письмо, содержащее из 7 слов

- Определите вероятность того, что письмо является спамом, исходя из тренировочного набора данных
- Вычислите F(«спам»)
- Вычислите F(«не спам»)
- Вычислите вероятность P(Класс = «спам» | Письмо)
