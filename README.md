**task 5.1**

Представим ситуацию, что из-за невнимательности операциониста, регионы: Камчатский край, Ивановская область, Хабаровский край оказались не представлены в итоговой сводке. Роль невнимательного операциониста придется исполнить Вам (т.е. нужно удалить соответствующие строки), а далее работать уже с новой выборкой.

Рекомендуем получив выборку воспользоваться RapidMiner или Python для определения статистик и знакомства с инструментами.

Постройте вариационный ряд X(сортировка по неубыванию, нумерация элементов начинается с 1).
- Ввести $Х_4$
- Ввести $Х_{22}$
